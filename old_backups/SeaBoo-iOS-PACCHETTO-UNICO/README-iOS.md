## 🚀 Build per iOS:

```bash
# 1. Build frontend
npx vite build

# 2. Sincronizza iOS
npx cap sync ios

# 3. Apri Xcode
npx cap open ios
```

## 🔧 In Xcode:
1. Seleziona Team Apple Developer
2. Cambia Version: 1.0.1
3. Cambia Build: 2  
4. Target: "Any iOS Device"
5. Product → Archive
